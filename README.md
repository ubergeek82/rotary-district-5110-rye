# Rotary District 5110 Youth Exchange Website

Official website for Rotary District 5110 Youth Exchange programs.

## Project Setup

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

The site is deployed to GitHub Pages at https://www.rye5110.org

To deploy changes:
1. Build the production version: `npm run build`
2. Copy built files to root: `cp -r dist/* .`
3. Commit and push to main branch
